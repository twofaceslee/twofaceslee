깃헙

이 프로젝트는 오픈소스로 운영되고 있습니다.

GitHub를 통해 프로젝트에 기여하실 수 있습니다:

github.com/twofaceslee/twofaceslee

기여 방법

  1. 저장소 포크하기

    GitHub 저장소 우측 상단의 'Fork' 버튼을 클릭하여 자신의 계정으로 프로젝트를 복제합니다.

  2. 로컬에 클론하기

    포크한 저장소를 로컬 컴퓨터로 클론합니다:

    git clone https://github.com/[사용자명]/twofaceslee.git
  3. README.md 확인하기

    새로운 섹션 추가 방법과 가이드라인은 README.md 파일을 참고해 주세요.

    섹션 추가 시 반드시 README.md의 가이드라인을 따라주세요.

  4. 브랜치 생성하기

    새로운 기능이나 수정사항을 위한 브랜치를 생성합니다:

    git checkout -b feature/your-feature-name
  5. 변경사항 커밋하기

    변경사항을 커밋합니다:

    git add . git commit -m "설명"
  6. 푸시하기

    변경사항을 GitHub에 푸시합니다:

    git push origin feature/your-feature-name
  7. 풀 리퀘스트 생성하기

    GitHub에서 'Pull Request'를 생성하여 변경사항을 제안합니다.

    중요: PR 설명에 다음 내용을 반드시 포함해주세요:

    • 수정한 내용의 출처 (기사, 공식 발표 등)
    • 관련 링크나 참고 자료
    • 수정 이유와 근거

※ 모든 기여는 반드시 신뢰할 수 있는 출처를 바탕으로 해야 합니다.

※ 출처가 없는 PR은 승인되지 않습니다.
